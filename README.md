# INST326_final
The INST326 final project
This project was aimed at simplifying and making it easier for people to organize their bills or invoices. This is done through a python program that is 
callable through the console. The console provides different options to the end user that allows them to decide what they want to do. These optioens include creating a new collection of bills that would replace the old one, adding new bills, checking the balance, paying off bills, and getting a list of bills with their information. Each bill has a unique identifier related to it so that it does not end up being confused with another bill.

The bills are saved as a class through a pickle file that's created when the first collection of bills is made.
